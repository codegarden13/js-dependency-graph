# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

We are committed to providing a respectful and inclusive environment for everyone.
Harassment, discrimination, and abusive behavior are not tolerated.

Project maintainers are responsible for enforcing this code of conduct.
Violations may result in temporary or permanent exclusion from the project.

For details, see:
https://www.contributor-covenant.org/version/2/1/code_of_conduct.html
